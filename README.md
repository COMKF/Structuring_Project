这个工程有两个目的：
-------------
### 阅读源码，记录笔记。
     源码的阅读方式是：
    （1）选择一个合适的工程（自己能用的到的），然后从github上下载下来。
    （2）尽可能从最低的版本（一般是v0.1）开始阅读，仔细阅读，尽量理解每一行代码，然后不断的提高阅读的版本。
    （3）对于源码的功能，要理解透，对于源码引入的第三方包，只要明白它的功能就行了，可以不深究。

### 学习将python工程结构化，写入优雅的python代码。
    学习各种工具，并找到最佳的实现方式。
